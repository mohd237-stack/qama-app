# ملفات منصة صيف القمع المحدثة

ارفع **جميع محتويات هذا المجلد** إلى جذر مستودع GitHub، وليس ملف index.html فقط.

الملفات المهمة:
- `index.html`: الموقع المحدث.
- `manifest.webmanifest`: معلومات تثبيت المنصة كتطبيق ويب.
- `sw.js`: خدمة التثبيت والتخزين المؤقت.
- `_headers`: إعدادات التحديث الصحيحة في Netlify.
- `assets/`: صورة صيف القمع ورمز QR.
- `icons/`: أيقونات التطبيق.
- `index-before-services.html`: نسخة احتياطية من الموقع السابق؛ لا يلزم نشرها ويمكن حذفها من GitHub بعد التأكد.

بعد الرفع:
1. انتظر اكتمال Deploy في Netlify.
2. افتح الموقع في نافذة خاصة أو امسح ذاكرة المتصفح للتأكد من النسخة الجديدة.
3. من Netlify افتح Forms وتأكد أن Form detection مفعّل ليستقبل نموذج `error-report`.
4. الإحصاءات التفصيلية اختيارية: أنشئ GA4 ثم ضع المعرّف في `GA_MEASUREMENT_ID` داخل index.html.
